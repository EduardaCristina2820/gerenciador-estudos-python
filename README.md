# Gerenciador Inteligente de Estudos em Python

Um sistema desenvolvido em Python para auxiliar estudantes no planejamento, acompanhamento e análise de seus estudos de forma simples e eficiente, diretamente pelo terminal.

---

## Informações sobre o projeto 

Este projeto foi criado com o objetivo de resolver um problema real: **organizar e monitorar o tempo de estudo de forma prática**.

A aplicação permite ao usuário cadastrar matérias, definir metas semanais e acompanhar o progresso através de sessões de estudo registradas. Tem a finalidade de possibilitar que os usuários identifiquem falhas e pontos de melhorias na rotina de estudos, ajudando, assim, no seu desenvolvimento acadêmico.   

Além disso, o sistema utiliza armazenamento em arquivos JSON, garantindo persistência de dados entre execuções — simulando o funcionamento de sistemas reais.

---

## Funcionalidades

- Cadastro de matérias
- Definição de metas semanais (horas e minutos)
- Cronômetro automático para registro de estudos
- Histórico de sessões de estudo

## Como Executar o Projeto

1. Clone o repositório


2. Acesse a pasta:

```
cd gerenciador-estudos-python
```

3. Execute o programa:

```
python main.py
```
